# cECG 非接触电极改板设计清单

## 1. 设计决策

| 项目 | 决定 |
|------|------|
| 总接头(线缆↔主板) | DP9 (D-Sub 9针，母座在主板/公头在线缆) |
| 电极板↔线缆接头 | JST (P/N电极板5P，RLD电极板2P) |
| 屏蔽方案 | 独立双屏蔽 (P/N各一路 driven-shield) |
| 电极模式切换 | 主板0Ω焊桥，电容/有源 vs 接触验证 |

---

## 2. 接头方案

### 2.1 电极板↔线缆: JST

| 板子 | JST针数 | 信号 |
|------|---------|------|
| P/N电极小板(各一个) | 5P (建议JST SH 1.0mm) | VDDA, V_BIAS, GND, SIG, SHIELD |
| RLD电极板 | 2P | RLD, GND |

小板体积小，优先贴身端用JST而非直接焊线缆，方便迭代时插拔更换电极板。

### 2.2 线缆↔主板: DP9 (D-Sub 9针)

**重要: D-Sub物理上是上5下4两排交错，不是一排直线9针。** 針位必须按物理相邻关系重新映射，否则"SIG两侧夹SHIELD/GND隔离"的用意会落空。

```
 1   2   3   4   5      ← 上排(5针)
   6   7   8   9        ← 下排(4针，卡在上排相邻两针中间下方)
```

物理相邻路径: 1-6-2-7-3-8-4-9-5 (下排每针卡在上排相邻两针中间)

**DE-9针位表(按物理相邻关系映射):**

| DE-9针脚 | 信号 | 方向 | 说明 |
|----------|------|------|------|
| 1 | VDDA | 主板→电极 | 两颗OPA391供电(~50µA)，远离信号 |
| 2 | SHIELD_P | 电极驱动 | P路driven-shield |
| 3 | GND | 共用 | 物理上同时挨着7(SIG_P)和8(SIG_N)，居中隔离 |
| 4 | SHIELD_N | 电极驱动 | N路driven-shield |
| 5 | GND2 | 共用 | 整束外层屏蔽回流，远离信号 |
| 6 | V_BIAS | 主板→电极 | VDDA/2偏置基准，远离信号 |
| 7 | SIG_P | 电极→主板 | 物理上夹在SHIELD_P(2)和GND(3)之间 |
| 8 | SIG_N | 电极→主板 | 物理上夹在GND(3)和SHIELD_N(4)之间 |
| 9 | RLD | 主板→电极 | 驱动电极，RLDOUT经R22(100k)送出，远离信号 |

按物理路径验证隔离: VDDA(1)→V_BIAS(6)→SHIELD_P(2)→**SIG_P(7)**→GND(3)→**SIG_N(8)**→SHIELD_N(4)→RLD(9)→GND2(5)，两路SIG依然被SHIELD/GND物理夹住。

**壳体接地**: 线缆整束外层编织屏蔽接D-Sub**金属壳体**(不塞进某个信号针脚)，主板内部壳体地与GND2(pin5)单点星形连接，避免地环路。

**公母**: 主板装母座(插针内凹)，线缆端做公头，防止设备静置时裸针脚被意外碰触短路。

## 3. 电极端小板设计 (P路，N路镜像)

### 信号流
```
金属电容板(背覆Kapton/阻焊做介质)
  → 高阻节点
    → 100MΩ电阻到V_BIAS (偏置)
    → OPA391 单位增益跟随
      → 输出分两路:
        ├─ 经~100Ω串阻 → SIG_P (回主板)
        └─ 直接驱动 SHIELD_P (guard ring + 线缆内屏蔽)
```

### 关键要点
- **100MΩ偏置电阻 + guard ring 全部留在电极端小板上**
- 高阻节点绝不出小板 —— 有源电极的核心意义
- OPA391供电就近加100nF去耦 (参照原C26/C27)
- Guard ring环绕高阻节点走线，driven电位 = OPA391输出
- 电极小板尽量小、尽量靠近金属板，减少高阻节点走线暴露

### 电容板与小PCB的连接方式（不要用"焊pin+飞线"）
- **不推荐**: 小板上焊一个pin再用飞线接到电极板 —— 飞线完全暴露在guard ring之外，等于给全板最敏感的高阻节点接了根天线；两处焊点还带来助焊剂残留漏电风险（几十~几百MΩ级残留电阻会和100MΩ偏置电阻抢电流，破坏偏置精度、引入低频漂移）
- **推荐(最优)**: 电容板直接是小PCB最外层的一块铺铜/贴片金属，背面覆Kapton或加厚阻焊做介质层。板到100MΩ/OPA391全程走内部PCB线，被guard ring全程包裹，零飞线零额外焊点
- **推荐(次优)**: 若金属板需单独做(如不锈钢片更耐用)，直接贴焊在PCB大焊盘上(面接触，零长度过渡)，不经过飞线
- **若结构上必须留一小段距离**: 用短同轴线(屏蔽层接guard/OPA391输出而非GND)，长度压在1cm内，不要用裸线
- 高阻焊点焊接后必须酒精清洗助焊剂残留 + 三防漆(conformal coating)保护，这一步对100MΩ节点是必须项而非可选

### Guard ring 连接规则（重要，防止误接）
- **小板本地**: guard ring只接OPA391输出引脚（极短走线，同一网络）；若双层板走高阻线，顶/底层都做ring，用过孔缝合ring自身，不得碰GND
- **延伸到线缆**: guard ring在PCB边缘直接接到该路单芯屏蔽线的屏蔽层导体（即SHIELD_P/SHIELD_N），本质是同一张被驱动等电位罩换了载体
- **主板/DP9这端 (Pin3 SHIELD_P / Pin7 SHIELD_N)**: **悬空NC，绝不接主板GND或任何其他网络**。接GND会让driven-shield退化成普通接地屏蔽，前功尽弃；两端电位不一致还会在屏蔽导线里产生持续电流，增加OPA391负载。可留测试点调试，正常应看到与SIG_P/N几乎同幅同相的波形

### RLD电极端
- 不需要缓冲
- 主板RLDOUT → R22(100k) → Pin9 → RLD电极板
- **RLD板务必做大**，耦合越强共模抑制越好（之前小RLD是问题根源）

### 3.1 P路原理图定稿 (2026-07, N路镜像)

| 元件 | 值 | 连接 |
|------|-----|------|
| U6 | OPA391 | IN+ = `E_PLATE_P`，IN−与OUT相连(跟随器，网络`Guard Ring P`)，V−=GND，V+=VDDA |
| R18 | 100MΩ | V_BIAS ↔ `E_PLATE_P` (偏置) |
| R19 | 1kΩ | OUT ↔ SIG_P 串阻 (100Ω更常规，1k可用，与主板10k抗混叠串联影响可忽略) |
| C27 | 100nF | VDDA–GND 就近去耦 |

- 高阻网络命名为 `E_PLATE_P`（不要留EDA自动匿名网络名如`$3N17`，改动后会被重新编号）
- **IN+节点上绝无GND**——曾出现过IN+误接GND的错误，检查时重点核对
- 电极连接：`E_PLATE_P` 经**一个盖油过孔**通到底层电极铺铜，不放裸露测试点（裸TP焊盘=高阻节点漏电口）
- OUT→Guard Ring P之间可留0Ω位（防长线缆屏蔽振荡，见第5节），线缆<50cm可省

### 3.2 小板PCB落地要点 (两层板：底层电极、顶层元件)

- **底层整层留给电极铜箔，任何其他信号严禁打过孔到底层**——GND等信号过孔下去要么短路电极、要么在电极里挖隔离岛并把干扰源引到电极正下方。全板唯一通底层的过孔就是`E_PLATE_P`那一个
- 高阻节点"领土" = U6 IN+焊盘 + 走线 + 过孔 + R18高阻侧焊盘，guard ring把这整片围住
- **Guard ring画法(立创EDA)**：闭合走线会被"自动移除回路"功能删除。三个办法：①设置→PCB→布线里关掉"移除回路"；②**用铺铜/实心填充画ring(推荐)**，网络设`Guard Ring P`，面对象不触发回路移除且拦截面积更大；③留0.3mm小口的C形ring，缺口朝板边无铜方向
- Guard ring在R18处从电阻本体下方两焊盘间穿过：高阻侧焊盘围在圈内，V_BIAS侧留圈外
- R18尽量贴近U6，缩小高阻区，guard圈小而完整
- U6相邻引脚(IN+与GND，SC70间距0.95mm)之间guard插不进去，物理极限，围好封装外部即可
- 高阻过孔盖油(tented)；贴皮肤板建议高阻区加三防漆——汗液潮气下FR4表面电阻掉到10⁸~10⁹Ω，与100MΩ偏置同数量级
- **走线长短不免除guard**：漏电取决于相邻铜的电位差与间距，不取决于走线长度；1mm走线+焊盘+孔环的暴露铜照样要围

### 3.3 电极材质结论

- **电容式电极不在乎金属材质**：铜被阻焊介质盖住，无电化学界面，铜/银/金电学等价。电容电极板=普通铺铜+阻焊(不开窗)，阻焊层本身就是介质，零成本最优解
- PCB导体层只能是铜，"换成银"只能做表面镀层(沉银)；且**沉银≠Ag/AgCl**——低漂移靠的是银/氯化银氧化还原电对(非极化电极)，纯银面依然是极化电极，贴皮漂移与普通金属差别不大
- 若需开窗露铜(接触焊盘等)：立创EDA在底层阻焊层画实心填充=开窗(阻焊层负片逻辑)；开窗面下单选**沉金**(平整耐汗液)，默认喷锡不平易氧化
- 接触验证模式直接用一次性Ag/AgCl凝胶电极片接到主板焊盘，不在PCB上折腾镀层

## 4. 主板侧改动

### 4.1 引出到DP9的信号
- VDDA、V_BIAS、GND 引到DP9 (原V_BIAS只在板内)

### 4.2 原缓冲级处理
- 原U4/U6 (OPA391) 缓冲级已移到电极端 → 主板上去掉
- 保留位置做接触/电容切换焊桥

### 4.3 信号进ADS前的0Ω二选一焊桥 (IN2P/IN2N各一组)

**模式A: 电容/有源模式**
```
DP9的SIG_P/SIG_N → 10kΩ(R26/R27) + 100pF抗混叠 → IN2P/IN2N
```

**模式B: 接触验证模式**
```
临时接触电极焊盘(Ag/AgCl)
  → 串联~0.1µF交流耦合
  → 几MΩ偏置到V_BIAS
  → 同一IN2P/IN2N
```

- 同一时刻只焊一侧
- RLD两模式共用

### 4.4 输入ESD保护
- DP9的SIG_P/SIG_N进板处加低漏电钳位
- 参照RLD那颗BAV199，注意选低漏电型号
- SHIELD线不需要钳位

## 5. 稳定性注意点

OPA391直接驱动有电容的屏蔽/guard属于容性负载：
- OPA391数据手册标称可稳定驱动高达**1nF**容性负载
- 短引线(几十cm级)一般没问题
- 如果线拉得很长、屏蔽电容大 → 在OPA391输出到屏蔽之间串小电阻(几十~100Ω)隔离，防振荡
- **留个位子，先不焊**

## 6. 验证步骤顺序

1. **先焊接触验证模式** (模式B)
   - 贴标准Ag/AgCl电极片
   - 电极在身体上拉开跨过心轴
   - 验证链路+几何能出QRS

2. **确认后切回有源电极** (模式A)
   - 焊接0Ω到电容/有源侧
   - 专测电容前端

3. **对比两种模式波形**
   - 确认有源电极改善效果

## 7. 线缆建议

**不是"一根两芯共用屏蔽"，而是两根各自独立屏蔽的单芯线** —— 每路信号自己的屏蔽层就是它的driven-shield，P/N不能共用屏蔽（共用会导致其中一路guard电位失配，等于白做）。

线材构成：
```
P路: 单芯屏蔽线×1 — 芯线=SIG_P，屏蔽层=SHIELD_P (接OPA391_P输出)
N路: 单芯屏蔽线×1 — 芯线=SIG_N，屏蔽层=SHIELD_N (接OPA391_N输出)
公共线: 4根普通导线(不需独立屏蔽) — VDDA, V_BIAS, GND, RLD
外层: 整束再包一层公共屏蔽/编织层 → 接D-Sub金属壳体，壳体与GND2(Pin5)在主板单点星形连接 (整体抗外部耦合，非二次guard)
```

- 现成线材可选"逐芯独立屏蔽多芯线"(如Belden 8723类构型：双芯各自屏蔽+公共屏蔽)
- 或自制：2根细同轴线(各自屏蔽) + 4根普通线，同走一根护套
